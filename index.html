CodeCrunch Countdown

A productivity timer for developers with built-in git commit reminders and debug session tracking.

Features

  • Pomodoro Timer - Focus for 25 minutes, then take a 5-minute break
  • Git Commit Reminder - Get reminded to commit your changes regularly
  • Debug Session Timer - Timebox your debugging sessions
  • TODO List - Track bugs and tasks in the debug mode
  • Productivity Stats - Track your focus sessions, bugs fixed, and coffee consumed

Getting Started

  1. Select a timer type from the sidebar
  2. Click Start to begin the timer
  3. Take breaks when the timer completes
  4. Track your progress in the status bar

Timer Types

Pomodoro

The classic 25-minute focus timer with a 5-minute break. Adjust durations in Settings.

Controls: Start, Pause, Reset, +1min

Git Timer

Reminds you to commit your changes every 30 minutes (configurable). Shows progress toward next commit.

Controls: Start, Pause, Reset

Debug Mode

Timebox your debugging sessions to stay focused. Includes a TODO list to track bugs.

Controls: Start, Pause, Reset, Skip, +1 Coffee

TODO List

In Debug Mode, you can:

  • Add new bugs/tasks to fix
  • Check off completed items
  • Each checked item increments your "Bugs Fixed" counter

Keyboard Shortcuts

  • Ctrl+Shift+C - Add coffee (debug mode)
  • Alt+F4 - Pause current timer (joke)
  • Ctrl+Shift+Enter - Konami code (bug fix boost)

Settings

Customize timer durations and view stats in the Settings tab.

Pro Tip: The timer will automatically save your settings and stats between sessions.
